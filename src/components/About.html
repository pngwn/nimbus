nimbus

weather made simple

nimbus forecasts the weather in a given location for today and tomorrow only. The day is split into four six-hour blocks: morning (0600 - 1200), afternoon (1200 - 1800), evening (1800 - 0000), night (0000 - 0600)

Each block is based on an hourly forecast. The icons show all of the weather types you can expect during each six-hour block, if different types of precipitation are forcast only the worst will be shown (rain > hail > snow). The listed temperatures and wind speed will be the most extreme forecast for that period, humidity will be the average. The summary will list the most likely and common expected weather for that period.

This is an experiment and might be wrong. If you find yourself up shit creek wearing the wrong kind of socks, go find someone else to blame.

nimbus is built with Svelte and Sapper and powered by Netlify and Google Cloud Functions. It uses the Dark Sky API for weather data and the Google Places API for geocoding and location autocomplete. The source code is availble on github.
